# 文字練習アプリ

半透明グレーの見本文字の上をなぞって、文字や文章の練習ができるWebアプリです。

## 使い方

1. 「練習する文字」に、練習したい文字や文章を入力します
2. 「文字を反映」ボタンを押すと、練習エリアに見本が表示されます
3. マウスや指で、グレーの文字の上をなぞって練習します
4. 「なぞりを消す」で書いた線を消せます。「ひとつ戻す」で直前の線だけ戻せます

## ローカルで試す

ブラウザで `index.html` を開くだけで動作します。

```bash
# 簡易サーバーで起動する場合
python3 -m http.server 8080
```

## GitHub Pages

`main` ブランチにマージされると、GitHub Actions により自動デプロイされます。

公開URL: https://ishitanikatuya.github.io/writeapt/

> **初回のみ**: リポジトリの Settings → Pages で、Source を **GitHub Actions** に設定してください。

## 機能

- 自由入力の文字・文章を見本として表示
- 見本の文字サイズ・濃さを調整可能
- ペンの太さを調整可能
- マウス・タッチ（スマホ・タブレット）対応
- なぞりの取り消し・クリア
